Database Type

This app is using MongoDB, a NoSQL database type, which stores data records as BSON documents. BSON is a binary representation of JSON documents. The MORPG database is hosted by the mLab cloud services provider.

Database Structure

The general structure and Entity Relationships of MORPG Database can be seen on the image below. A PDF document is also available. The one-to-many relations are handled using both the Embedded (De-normalized) Method as well as the Relational (Normalized) Method, in an attempt test both their strengths.

The User document includes the Character sub-document. The Weapon and Type documents are referenced in the Character sub-document. The Location document is referenced in the Type document and it includes the Monster sub-document.
